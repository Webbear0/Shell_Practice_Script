# Shell_Practice_Script 学习项目

> 这是一个为辅助学习 bash shell 编程而编写的脚本集合项目。项目包含日常系统管理、服务搭建和逻辑处理练习等多种实用脚本。

---

##  脚本目录与功能说明

本项目共包含 6 个核心 Shell 脚本，功能涵盖数据库操作、环境部署及系统信息抓取等：

### 1. `osinfo.sh` (系统信息查看工具)
- **功能**: 提供交互式菜单，用于查看当前系统的 CPU、网络、内存、进程、磁盘以及系统用户信息。
- **特点**: 包含大量的代码注释（涉及变量定义、条件判断、环境变量、重定向等知识点），非常适合初学者对照学习 Bash 的基础语法。

### 2. `LAMP.sh` (LAMP 一键部署脚本)
- **功能**: 提供菜单引导，实现一键安装 Apache、PHP、MariaDB、phpMyAdmin、WordPress 和 Discuz。
- **特点**: 包含了服务自动放行防火墙、开机自启配置、源码包解压安装及目录权限配置等生产环境常见操作。
- **执行要求**: 需要 `root` 权限。

### 3. `mariadb.sh` (MariaDB 自动安装脚本)
- **功能**: 精简版的数据库自动安装工具，执行后自动配置 MariaDB 服务并设定 root 密码，同时支持可选安装 phpMyAdmin。
- **执行要求**: 需要 `root` 权限。

### 4. `bank_os.sh` (银行业务管理系统)
- **功能**: 基于 MySQL/MariaDB 数据库实现的控制台银行管理系统。支持：
  - 数据库/数据表一键初始化
  - 用户开户与销户
  - 模拟存款与取款（带余额校验逻辑）
  - 余额实时查询
- **特点**: 演示了如何通过 Bash 脚本连接 MySQL 数据库，并使用 Heredoc (`<<EOF`) 在脚本中批量执行 SQL 语句。

### 5. `yumrepos.sh` (YUM 本地源一键配置)
- **功能**: 将 CentOS 默认的网络 YUM 源一键替换为基于光驱 (`/dev/cdrom`) 的本地源。
- **特点**: 包含了系统设备的检测逻辑、配置文件的自动备份、以及通过 `sed` 命令对配置文件的流式修改。
- **执行要求**: 需要 `root` 权限及已挂载的光驱设备。

### 6. `motd.sh` (动态登录 Banner)
- **功能**: 打印自定义格式的系统登录欢迎语 (Message of the Day)，包含当前主机名及系统版本信息。
- **特点**: 演示了通过 `printf` 对齐字符串以及动态命令替换 (`$(command)`) 的技巧。

##  使用说明

1. **赋予执行权限**:
   在运行任何脚本前，请确保文件拥有可执行权限：
   ```bash
   chmod +x *.sh
   ```

2. **以 Root 身份运行**:
   涉及系统配置修改或环境安装的脚本（如 `LAMP.sh`、`mariadb.sh`、`yumrepos.sh`）内置了权限检测逻辑，必须以 `root` 用户身份执行。

3. **运行脚本**:
   ```bash
   ./脚本名称.sh
   # 例如: ./osinfo.sh
   ```
